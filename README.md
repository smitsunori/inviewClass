# inviewClass
js scroll helper

Drafting...

## Overview
- add class when item enter
  - add second class on defined interval
 
## Usage
- init
$('.your-item').inviewClass('is-inview');



### options

- offset
- secondClassInterval

$('.your-item').inviewClass({
  className: 'is-inview' //set to change classname (optional)
  offset: 100, //px
  secondClassInterval: 1000 //ms
});
