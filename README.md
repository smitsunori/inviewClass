## Overview
- add class when item enter
- toggle class when item enter/leave
  - add class when item leave?
  - adjust scroll offset
 
## Usage
- init

$('.your-item').inviewClass();



### options

- offset
- secondClassInterval

$('.your-item').inviewClass({
  offset: 100, //px
  secondClassInterval: 1000 //ms
});
